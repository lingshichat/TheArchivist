# 阶段1-WP3：Windows 首页、库页、详情页骨架

## 目标

把 Windows 首版的主界面和核心页面骨架搭起来，形成可演示桌面工作面。

## 范围

* 左侧导航
* 首页布局
* 媒介库页布局
* 详情页双栏布局
* 海报网格基础组件
* 空状态占位

## 不包含

* Bangumi 搜索
* 跨端同步
* 复杂筛选器

## 验收标准

* [ ] 首页包含继续中、最近添加、最近完成、媒介入口
* [ ] 库页有媒介切换和海报网格
* [ ] 详情页双栏信息层级清楚
* [ ] Windows 端不是移动页面简单拉宽

## 依赖

* 依赖 `04-19-phase1-wp1-foundation-theme`
* 与 `04-19-phase1-wp2-local-data` 联调
