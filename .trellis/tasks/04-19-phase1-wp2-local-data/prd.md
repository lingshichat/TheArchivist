# 阶段1-WP2：本地数据模型与 Drift

## 目标

建立首版本地数据层，让首页、库页、详情页能消费稳定的本地记录数据。

## 范围

* `drift` 初始化
* 本地数据库入口
* `MediaItem`
* `UserEntry`
* `ProgressEntry`
* 标签、列表的最小表结构
* 基础查询与写入仓储

## 不包含

* 外部平台字段同步逻辑
* WebDAV / S3-compatible 同步
* 冲突处理

## 验收标准

* [ ] 本地库可创建、查询、更新、删除核心记录
* [ ] 状态、评分、进度、私人笔记可独立持久化
* [ ] 为外部 ID 与同步字段保留扩展位
