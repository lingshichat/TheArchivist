# 阶段4：设置、备份恢复与同步运维

## 目标

把同步与备份能力做成用户能自行配置、查看和恢复的完整操作面。

## 范围

* 设置页同步入口
* `WebDAV` / `S3-compatible` 配置表单
* 连接测试
* 快照包导出与导入
* 待同步列表
* 同步失败重试
* 冲突副本查看

## 不包含

* 多账号体系
* 复杂团队协作运维能力

## 关键交付

* 用户可以不依赖开发者手动排查，自行完成同步配置和备份恢复

## 验收标准

* [ ] 设置页可配置并保存同步目标
* [ ] 同步目标支持连接测试
* [ ] 可手动导出和导入快照包
* [ ] 可查看待同步项目并手动重试
* [ ] 可查看冲突副本并知道需要人工处理

## 依赖

* 依赖 `04-19-phase2-bangumi-flow`
* 依赖 `04-19-phase3-sync-engine`
