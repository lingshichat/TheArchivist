# 阶段5：桌面适配、性能与验收

## 目标

把产品从“功能成立”收成“日常可长期使用”。

## 范围

* Windows 双栏 / 左侧导航适配
* 鼠标悬停与桌面交互优化
* 海报墙滚动性能优化
* 图片缓存与解码优化
* 空状态、错误状态完善
* 回归测试与验收清单

## 不包含

* 新功能扩展
* 多源搜索
* 新外部平台接入

## 关键交付

* Android 和 Windows 两端都达到稳定可用的交付线

## 验收标准

* [ ] Windows 布局不再是移动端简单拉宽
* [ ] 海报墙在大量数据下仍保持流畅
* [ ] 主要空状态、错误状态、同步失败状态都有明确反馈
* [ ] 关键主链路通过手动回归
* [ ] 首版发布范围与未完成项有明确边界
