# 阶段1-WP1：工程骨架与主题令牌

## 目标

完成 Flutter 工程初始化，建立 Windows 优先的应用壳、主题系统和基础设计令牌。

## 范围

* Flutter 项目初始化
* 基础目录结构
* `go_router` 路由骨架
* 应用壳与导航骨架
* 主题、色板、圆角、间距、文字层级令牌

## 不包含

* 业务数据表
* Bangumi 接入
* 同步逻辑

## 验收标准

* [x] Windows 端能启动并进入应用壳
* [x] 至少有首页、媒介库、设置三个路由占位
* [x] 主题令牌可被页面统一消费
* [x] 页面不是默认 Flutter 脚手架视觉

## 依赖

* 无
