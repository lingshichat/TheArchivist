# 阶段2：Bangumi 搜索、快捷添加与同步

## 目标

打通“Bangumi 搜索 → 快捷添加 → 本地保存 → 自动同步”的核心使用体验。

## 范围

* Bangumi 单源搜索
* 条目元数据映射与入库
* 搜索结果卡片直接添加
* 添加前状态选择层
* Bangumi 可选绑定
* 状态、评分、进度同步

## 不包含

* 多源聚合搜索
* WebDAV / S3-compatible 跨端同步
* 冲突副本查看界面

## 关键交付

* 用户在应用内搜到 Bangumi 条目后，可快速加入本地库
* 已绑定 Bangumi 时，新增记录和后续状态改动会自动同步

## 验收标准

* [ ] 搜索结果带出封面、标题、年份、媒介类型、基础简介
* [ ] 点击“添加”后，先弹状态选择层
* [ ] 保存后立即写入本地库
* [ ] 已绑定 Bangumi 时，状态、评分、进度修改默认自动同步
* [ ] 同步失败时，本地记录保留并标记“待同步”

## 依赖

* 依赖 `04-19-phase1-local-core`

## 额外说明

* 进度同步要按媒介类型分流，不允许把所有进度都走同一接口
* 私人笔记、自定义标签、自定义列表不回写 Bangumi
