# Frontend Development Guidelines

> Best practices for frontend development in this project.

---

## Overview

This directory contains the working frontend conventions for the Windows-first
Flutter application at repo root.

The visual source of truth for Phase 1 is the Stitch-generated "The Archivist"
desktop design, not generic Material defaults.
